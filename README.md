# portafolio-personal
portafolio 
**INSTALAR NODE EN PC LOCAL
-> Descarga el motor nodejs desde su pagina oficial www.nodejs.org
-> Instalar node js en el equipo local

INSTALA MOTOR DB
-> Descarga XAMPP desde su pagina oficial www.apachefriends.org
-> Instala XAMPP en el equipo local
-> Ejecuta motor MySQL en XAMPP

Clona el proyecto en el equipo local
-> Ejecutar en terminal "git clone http://www.github.com/karlo-co/proyecto-personal"
-> Ejecuta la solución con node desde la terminal "node server.js"

Entra al sitio web
->En su navegador web ingresar a la sig URL "http://localhost:3000"
